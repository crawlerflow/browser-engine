Introduction

Introduction to core concepts, setup, architecture & how to extend.

Setup Server

export SELENIUM_HOST=http://127.0.0.1:4444 # this is selenium server.
export BROWSER_TYPE=chrome
export AUTH_TOKEN=iamlazydeveloper

# running with uwsgi
uwsgi --socket 0.0.0.0:5000 --protocol=http -w browser_engine.server.wsgi:application --processes 4 --threads 2

# or

# running as simple python server
python -m  browser_engine.server.app run

Terminology

term Description
WebBrowser Web Browser can be simple python http client, selenium htmlunit, selenium chrome, selenium firefox or splash.
WebRequest Request to run a web simulation
WebResponse Response from web simulation
Task Task to run on top of the browser ie., get screenshot, get page source, extract json, submit form, run pythonic simulation with selenium driver.
